# Table Actions

Operations on the game table and zones. Place, move, arrange tokens in spatial layouts.

[← Back to Action Reference](../ACTIONS.md)

---

## Actions (13)

**Basic Operations**
1. `table:place` - Place token in zone
2. `table:clear` - Clear entire table
3. `table:move` - Move token between zones
4. `table:flip` - Flip token face up/down
5. `table:remove` - Remove token from table

**Zone Management**
6. `table:createZone` - Create new zone
7. `table:deleteZone` - Delete zone
8. `table:clearZone` - Clear specific zone

**Zone Operations**
9. `table:shuffleZone` - Randomize positions in zone
10. `table:transferZone` - Move all tokens between zones
11. `table:fanZone` - Arrange in fan layout
12. `table:stackZone` - Stack tokens vertically
13. `table:spreadZone` - Spread in a line
14. `table:lockZone` - Lock/unlock zone
